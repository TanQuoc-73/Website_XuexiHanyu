# Chinese Language Learning Resources (🐼 学习汉语)

This document contains a curated list of libraries, datasets, and tools to help build the Xuexi Hanyu project.

---

## 1. Vocabulary Datasets (HSK 1-6)
These are essential for populating our database with official Hanzi characters, Pinyin, and meanings.

- **drkameleon/complete-hsk-vocabulary** (JSON)
  - *Link:* https://github.com/drkameleon/complete-hsk-vocabulary
  - *Best for:* Complete, structured data (part of speech, frequency, etc.).
- **oetia/new-hsk-csv** (CSV)
  - *Link:* https://github.com/oetia/new-hsk-csv
  - *Best for:* Simple CSV import to PostgreSQL.
- **Hugging Face - willfliaw/hsk-dataset**
  - *Link:* https://huggingface.co/datasets/willfliaw/hsk-dataset
  - *Best for:* Clean data and analysis.

---

## 2. Dictionaries & Character Data
- **CC-CEDICT** (Chinese-English Dictionary)
  - *Link:* https://mxm.net.cn/?p=906 or https://www.mdbg.net/chinese/dictionary?page=cedict
- **Hanzi Writer Data** (Stroke Order Animation)
  - *Link:* https://hanziwriter.org/
  - *Best for:* Teaching users how to write characters correctly.
- **HanViet Dictionary** (For Vietnamese learners)
  - *Resource:* Search GitHub for `nduchieu/HanVietDictionary` to get Sino-Vietnamese readings.

---

## 3. Libraries & Tools (Frontend/Backend)
- **pinyin-pro** (JS/TS)
  - *Link:* https://www.npmjs.com/package/pinyin-pro
  - *Best for:* Real-time Hanzi to Pinyin conversion with tones.
- **Hanzi Writer Library** (JS)
  - *Link:* https://hanziwriter.org/docs.html
  - *Best for:* Displaying stroke animations on the website.
- **Jieba** (JS/Java/Python)
  - *Link:* https://github.com/yanyiwu/nodejieba (Node.js version)
  - *Best for:* Word segmentation (splitting sentences into meaningful words).

---

## 4. Flashcards & SRS
- **Anki Web (HSK Decks)**
  - *Link:* https://ankiweb.net/shared/decks/hsk
  - *Best for:* Finding curated community decks that can be exported to CSV.
